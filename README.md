## 🚀 Inicio Rápido

### Prerrequisitos

- Node.js (versión 16 o superior)
- npm o yarn

### Instalación

1. **Clona o copia el proyecto**
   ```bash
   # Si tienes git configurado
   git clone <repository-url>
   cd handyman-web
   ```

2. **Instala las dependencias**
   ```bash
   npm install
   ```

3. **Configura Tailwind CSS** (si es necesario)
   ```bash
   # Ya está configurado, pero puedes verificar
   npx tailwindcss init -p
   ```

4. **Inicia el servidor de desarrollo**
   ```bash
   npm run dev
   ```

5. **Abre tu navegador**
   - Visita `http://localhost:3000`
   - El sitio se recarga automáticamente al hacer cambios

### Scripts Disponibles

```bash
# Desarrollo
npm run dev          # Inicia servidor de desarrollo

# Producción
npm run build        # Construye para producción
npm run preview      # Previsualiza la construcción

# Calidad de código
npm run lint         # Ejecuta ESLint
```

## 🎨 Personalización

### Colores y Tema

Los colores están definidos en `tailwind.config.js`:

```javascript
colors: {
  cream: { /* Tonos crema */ },
  terracotta: { /* Tonos terracota */ },
  petrol: { /* Azul petróleo */ },
  sage: { /* Verde salvia */ }
}
```

### Contenido

- **Productos**: Edita `src/data/products.js`
- **Blog**: Modifica `src/data/blogPosts.js`
- **Testimonios**: Actualiza `src/data/testimonials.js`
- **Ferias**: Configura `src/data/ferias.js`

### Imágenes

Coloca las imágenes en la carpeta `public/img/` con la siguiente estructura:
```
public/img/
├── hero/           # Imágenes del carrusel principal
├── sweaters/       # Fotos de sweaters
├── gorras/         # Fotos de gorras
├── ponchos/        # Fotos de ponchos
├── blog/           # Imágenes para artículos
├── testimonials/   # Fotos de clientes
└── team/          # Fotos del equipo
```

## 🔗 Integración con Tienda Nube

Cada producto incluye un `linkTiendaNube` que redirige a la tienda externa:
